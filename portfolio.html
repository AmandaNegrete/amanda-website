Retriever Routes

Retriever Routes is a UMBC oriented navigation application website! The website features implementation of the Leaflet routing machine and Leaflet mapping system to create an interactive campus map.

The process was a collaborative one as we implemented the SCRUM methodology and JIRA board application to organize and complete the project, SRS documentation,and Sprint Reports within 3 months!
